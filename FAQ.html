เรียนทั้ง Hardware และ Software เช่น Embedded Systems, Network & IoT Protocol Cloud, Backend, Database Cybersecurity, AI for IoT

สาขานี้เหมาะกับคนที่ ชอบทดลอง ชอบดูว่าอุปกรณ์ทำงานยังไง สนุกกับการทำโปรเจกต์ ไม่ใช่แค่เรียนทฤษฎี ไม่กลัววงจรไฟฟ้า หรือพร้อมจะเรียนรู้มัน
อยากเข้าใจระบบทั้ง Hardware และ Software ไปพร้อมกัน ชอบคิดวิเคราะห์และแก้ปัญหาเป็นขั้นเป็นตอน

เป็นความร่วมมือพิเศษกับคณะวิทยาศาสตร์ น้องจะได้รับปริญญา 2 ใบ ได้ทั้งทักษะวิจัยเชิงลึกแบบนักวิทย์ และทักษะการประยุกต์ใช้งานจริงแบบวิศวกร
คุ้มสุดๆ สำหรับสาย Tech ยุคใหม่ ได้ วศ.บ. วิศวกรรมไอโอทีและสารสนเทศ และได้ วท.บ. ฟิสิกส์อุตสาหกรรม

เรียนไหวแน่นอน หากมีความตั้งใจและพร้อมฝึกฝนอย่างสม่ำเสมอ ในปีแรกของหลักสูตรวิศวกรรมไอโอทีจะเริ่มสอนพื้นฐานการเขียนโปรแกรมตั้งแต่ระดับเบื้องต้น

Computer Engineering → เน้น Software การเขียนโปรแกรม Hardware ระบบคอมพิวเตอร์ต่างๆ
IT → เน้น Software และ ระบบสารสนเทศ
IoT Engineering → ผสมวงจร อุปกรณ์จริง เครือข่าย ซอฟต์แวร์ (จะได้จับบอร์ด, ต่อวงจร, เขียน Firmware แล้วส่งข้อมูลขึ้น Cloud เองครบวงจร)

วิศวกรรมระบบไอโอทีเเละสารสนเทศ (1 ปริญญา) จะเน้นการเชื่อมต่อและการจัดการข้อมูล เรียนเหมือนเป็นจอมวางแผนระบบ คือเอาอุปกรณ์ที่มีอยู่แล้วมาเขียนโปรแกรม ให้คุยกันได้ เก็บข้อมูลขึ้น Cloud และสร้าง Dashboard มาแสดงผล

ฟิสิกส์อุตสาหกรรม และวิศวกรรมระบบไอโอทีเเละสารสนเทศ (2 ปริญญา) เน้นเครื่องมือวัดและความแม่นยำ เรียนเหมือนเป็นนักประดิษฐ์และปรับแต่ง คือต้องเข้าใจลึกไปถึงระดับอะตอมหรือคลื่นฟิสิกส์ เพื่อสร้างเซนเซอร์ขึ้นมาเอง หรือแก้ไขระบบวัดในโรงงานที่ซับซ้อนเกินกว่าเซนเซอร์ทั่วไปจะทำได้

หลักสูตรเราเน้นพื้นฐานคำนวณและฟิสิกส์ค่อนข้างแน่น หากน้องมีความตั้งใจและมีคะแนนสอบตามเกณฑ์ที่กำหนด (เช่น TGAT/TPAT) ก็สามารถสมัครได้ แต่อาจจะต้องฟิตพื้นฐานวิทย์คณิตเพิ่มเติมตอนเข้ามาเรียน

มีโปรเจกต์ให้ทำตั้งแต่ปี 1-4 จะมีโปรเจคในแต่ละวิชา, Team Project, และ Senior Project (โปรเจกต์จบ)

สำหรับการเรียนในรายวิชามีอุปกรณ์พื้นฐานในห้องปฏิบัติการ เช่น Microcontroller Board อุปกรณ์วัดทางไฟฟ้า และชุดทดลองต่าง ๆ สำหรับใช้ในการเรียนการสอน และสำหรับการฝึกเพิ่มเติมหรือทำโปรเจกต์ส่วนตัว อาจพิจารณาซื้อเองตามความเหมาะสมแต่สามารถยืมอุปกรณ์จากสาขาได้

1. จบ "วิศวกรรมระบบไอโอทีเเละสารสนเทศ"
เช่น IoT Engineer, Network Engineer, Full-Stack Developer, Data Engineer, Cybersecurity Engineer

2. จบ "ฟิสิกส์อุตสาหกรรม เเละ วิศวกรรมระบบไอโอที"
เช่น Instrumentation & Control Engineer, R&D Engineer, Metrology Specialist, Process Engineer, Optical Medical Engineer
